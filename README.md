## Installation

Install project

```bash
  git clone https://github.com/whayu901/myAlam.git
  cd myAlam
  yarn
```

## For IOS (install pod)

```bash
  cd ios && pod install && cd ..
```

## Running App android

```bash
yarn android
```

## Running App IOS

```bash
yarn ios
```
